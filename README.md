# 친구들의 타워 디펜스

Phaser.js, Socket.io, Node.js, Express로 만든 1v1 실시간 타워 디펜스 프로토타입입니다.

## 실행

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000`을 열면 됩니다. 첫 번째 접속자는 아래 진영, 두 번째 접속자는 위 진영으로 배정되며 두 명이 접속하면 자동으로 경기가 시작됩니다.

## 구현된 내용

- 킹타워 1개와 프린세스타워 2개씩 배치
- 킹타워 파괴 즉시 승리
- 3분 제한 시간, HP 동률 시 서든 데스
- 최대 10 엘릭서 자동 충전
- 8장 카드풀, 4장 손패, 사용 후 다음 카드 순환
- 덱 총합 코스트 40~55 검증
- 꽁호 1회용 필드 리셋
- 빼트맨의 여학생 카드 자동 추적 힐
- 유신 8명 동시 소환
- 미토스건휘 HP 50% 이하 자동 각성 및 각성 중 무적
- 박바둑 랜덤 광역 공격과 Friendly Fire
