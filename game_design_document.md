## 3. 게임 시스템

### 3-1. 타워 구조
- 각 플레이어는 **킹타워 1개 + 프린세스타워 2개** 보유 (총 3개)
- 타워는 사정거리 내 적 유닛을 자동 공격
- **킹타워가 파괴되면 즉시 패배**
- 프린세스타워 파괴 시 킹타워 공격 가능

### 3-2. 승리 조건
- 상대방 **킹타워를 파괴**하면 즉시 승리
- **시간 제한 3분** 종료 시 타워 HP가 많은 쪽 승리
- 동점일 경우 추가 시간 (Sudden Death)

### 3-3. 엘릭서 시스템
- 시간에 따라 **자동 충전** (Clash Royale 동일)
- **최대 10 엘릭서**
- 카드 소환 시 해당 코스트만큼 차감

### 3-4. 덱 시스템
- 전체 카드풀 **8장** 중 게임 시작 시 **랜덤으로 8장 덱 구성**
  > ※ 현재 카드풀 = 8장이므로 항상 전체 카드로 구성됨. 추후 카드풀 확장 시 랜덤 의미 증가
- 화면에는 **4장만 표시**
- 카드 1장 사용 시 **다음 카드가 자동으로 올라옴**
- 덱 총합 코스트 범위: **40~55**

### 🎴 짱가
| 항목 | 내용 |
|---|---|
| **코스트** | 5 |
| **타입** | 광역 딜러 (AoE) |
| **사정거리** | 중거리 |
| **이동속도** | 보통 |
| **HP** | 중간 |
| **공격력** | 중상 |
| **공격속도** | 느림 |

**스킬 — 음파 폭소 (Sonic Burst)**
- 전방 **부채꼴 범위**로 음파 광역 데미지
- 범위 내 적 전체에게 동시 데미지
- 스킬 시전 시 크게 웃으면서 소리 지르는 모션 재생
- 쿨타임 동안 단일 일반 공격

**외형**
- 키 크고 안경 쓴 여학생
- 머리를 하나로 묶은 포니테일
- 스킬 시 입 크게 벌리고 웃으며 절규하는 이펙트
- 음파 이펙트: 전방 부채꼴 파동

### 🎴 빼트맨
| 항목 | 내용 |
|---|---|
| **코스트** | 3 |
| **타입** | 힐러 서포터 |
| **사정거리** | 단거리 |
| **이동속도** | 빠름 |
| **HP** | 낮음 |
| **공격력** | 없음 (또는 매우 낮음) |
| **힐량** | 높음 |

**스킬 — 여학생 추적 힐 (Follow & Heal)**
- 필드의 **여학생 캐릭터를 자동으로 따라다니며** 지속 힐
- 여학생 캐릭터 없을 시 멍하니 대기 (힐 불가)
- 여럿일 경우 **가장 가까운 여학생**에게 붙음

**외형**
- 키 작고 마른 체형
- 삭발 (머리 밈)
- 운동 잘하는 느낌의 피부가 엄청 탄 남학생

### 🎴 박바둑
| 항목 | 내용 |
|---|---|
| **코스트** | 8 |
| **타입** | 리스크 딜러 (Chaos) |
| **사정거리** | 광역 |
| **이동속도** | 보통 |
| **HP** | 최상 |
| **공격력** | 매우 높음 |
| **공격속도** | 느림 |

**스킬 — 돌발행동 (What Are You Doing)**
- 갑자기 **예측불가한 이상한 행동** 모션 시전
- 범위 내 **적군에게 큰 데미지**
- 같은 범위 **아군에게도 약한 데미지** (Friendly Fire)
- 스킬 발동 타이밍 **랜덤**

**돌발행동 모션 예시 (랜덤)**
- 갑자기 드러눕기
- 혼자 빙글빙글 돌기
- 하늘 보고 소리치기

**외형**
- 통통한 체형
- 삭발 + 안경 쓴 남학생

### 🎴 꽁호
| 항목 | 내용 |
|---|---|
| **코스트** | 10 |
| **타입** | 필드 리셋 (Ascension) |
| **사정거리** | 전장 전체 |
| **이동속도** | 없음 (소환 즉시 기도) |
| **HP** | 낮음 |
| **공격력** | 없음 |
| **특수 규칙** | **1회용** — 한 게임에 딱 한 번만 사용 가능 |

**스킬 — 대승천 (The Rapture)**
1. 소환 즉시 **게임 화면 Freeze**
2. 꽁호가 전장 중앙으로 이동 후 **무릎 꿇고 기도 시작**
3. 하늘에서 **빛줄기 이펙트** 발생
4. 엄청난 빛의 임팩트와 함께 **전장의 모든 유닛 (아군 + 적군) 승천 소멸**
5. 꽁호 본인도 승천하여 사라짐
6. 결과: **필드 완전 초기화**

**연출 포인트**
- 기도 모션 중 찬송가 BGM 삽입
- 승천 시 캐릭터별 표정 다름
  - 짱가: 크게 웃으며 승천
  - 박바둑: 무표정으로 승천
  - 빼트맨: 당황한 표정으로 승천
  - 꽁호: 평온한 표정으로 승천

**외형**
- 안경 쓴 평범한 기독교 남학생

### 🎴 유신
| 항목 | 내용 |
|---|---|
| **코스트** | 4 |
| **타입** | 떼거지 근접 딜러 (Swarm) |
| **사정거리** | 근접 |
| **이동속도** | 빠름 |
| **HP (1명)** | 매우 낮음 |
| **공격력 (1명)** | 낮음 |
| **공격속도** | 빠름 |

**스킬 — 유신 군단 (Baby Punch Rush)**
- 소환 시 **유신 8명이 동시에 등장**
- 각자 근접 주먹질로 달려들어 공격
- 개별 스탯은 약하지만 **8명 총합 딜 강력**
- 한 명씩 처치될 때마다 줄어드는 구조

**외형**
- 키 작고 마르며 애기처럼 생긴 남학생
- 8명이 **똑같은 외형**으로 우르르 등장
- 소환 모션: 8명이 달려나오는 연출

### 🎴 지민
| 항목 | 내용 |
|---|---|
| **코스트** | 4 |
| **타입** | 단일 딜러 (Single Shot) |
| **사정거리** | 중거리 |
| **이동속도** | 보통 |
| **HP** | 중간 |
| **공격력** | 매우 높음 (단일 최강) |
| **공격속도** | 매우 느림 |

**스킬 — 필살 드립 (Punchline)**
- 공격 전 **드립 치는 모션** 시전 (이 동안 무방비 상태)
- 단일 타겟에게 **게임 내 최고 수준의 단일 데미지**
- 드립 모션 중 피격 시 공격 취소 가능

**외형**
- 키 크고 비율 좋은 남학생
- 드립 칠 때 손가락으로 가리키는 모션
- 맞은 적 위에 멘붕 이펙트

### 🎴 미토스건휘
| 항목 | 내용 |
|---|---|
| **코스트** | 5 |
| **타입** | 변신형 딜러 (Transform) |
| **사정거리** | 근접 |
| **이동속도** | 보통 → 빠름 (변신 후) |

**페이즈 1 — HP 50% 이상 (평상시)**
| 스탯 | 수치 |
|---|---|
| HP | 높음 |
| 공격력 | 낮음 |
| 공격속도 | 보통 |
| 이동속도 | 보통 |

**페이즈 2 — HP 50% 이하 (각성)**
| 스탯 | 수치 |
|---|---|
| HP | 동일 (회복 없음) |
| 공격력 | 매우 높음 |
| 공격속도 | 빠름 |
| 이동속도 | 빠름 |

**스킬 — 초싸이언 각성 (Awaken)**
- HP 50% 이하 도달 시 **자동 발동**
- 머리카락이 위로 솟구치는 **변신 모션** 재생
- 변신 모션 동안 **무적 판정**
- 한 번 변신하면 **되돌아오지 않음**

**외형**
- 평소: 얼굴 살짝 네모나고 안경 쓴 평범한 남학생, 머리카락 내려앉아 있음
- 변신 후: 머리카락 위로 쫙 올라가고 안경 날아가는 연출
- 변신 이펙트: 번개 + 빛 폭발

- 테니스채 들고 경쾌하게 휘두르는 모션
- 맞는 적 위에 🎾 이펙트

## 7. Claude Code 구현 지시사항

### 우선순위 순서
1. **기본 게임 루프** — 타워 배치, 엘릭서 충전, 카드 소환
2. **온라인 멀티** — Socket.io 기반 1v1 실시간 대전
3. **캐릭터 8종 구현** — 각 스탯 및 스킬 로직
4. **특수 규칙 구현**
   - 꽁호: 1회용 + 게임 Freeze + 전체 유닛 소멸
   - 빼트맨: 여학생 캐릭터 자동 추적
   - 유신: 8명 동시 소환
   - 미토스건휘: HP 50% 이하 자동 변신
   - 박바둑: Friendly Fire 처리
5. **UI/UX** — 카드 4장 표시, 엘릭서 바, 타워 HP 표시
6. **랜덤 덱** — 게임 시작 시 카드풀에서 8장 랜덤 구성

### 핵심 구현 주의사항
- 덱 총합 코스트 **40~55 범위** 검증 로직 포함
- 꽁호 **1회용** 처리 — 한 게임에 한 번 사용 후 덱에서 제거
- 빼트맨 힐 대상 — **여학생 캐릭터 (짱가, 복숭아)** 만 해당
- 박바둑 Friendly Fire — 아군 데미지는 적군 데미지보다 **현저히 낮게** 설정
- 미토스건휘 변신 모션 중 **무적 판정** 구현
