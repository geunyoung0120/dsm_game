# 업데이트 기록

이 파일은 지금까지 구현한 내용과 앞으로 추가되는 업데이트를 정리하는 문서다. 새 기능을 만들거나 기존 기능을 바꾸면 이 파일도 함께 갱신한다.

## 운영 규칙

- 코드 업데이트를 적용할 때마다 작업 후 `git add .`, 커밋, `git push`까지 진행한다.
- 업데이트를 적용할 때마다 메인 화면의 공지사항 게시판에 새 공지글을 추가한다.
- 메인 화면에는 최근 공지글 3개만 작게 보이고, `업데이트 목록 보기`에서 역대 공지글 전체와 자세히 보기를 확인하게 한다.
- 캐릭터 밸런스를 바꾸면 서버 전투 스탯과 캐릭터 확인 페이지 설명을 함께 갱신한다.
- Railway에 배포된 온라인 버전으로 확인할 수 있으므로, 업데이트 작업 마지막에 별도로 로컬 서버를 켜지 않는다.
